InsightLoop is a Python-based analytics tool that helps students track study habits, measure productivity, and build consistent learning streaks.

## Features
- Logs study sessions by subject
- Identifies most productive days
- Tracks longest study streak
- Visualizes progress toward study goals
- Provides structured productivity insights

## Tech Stack
Python, CSV, datetime

## Why I Built This
I wanted a lightweight system to better understand my learning patterns and stay accountable to long-term academic goals.

## Future Improvements
- Data visualization dashboard
- Machine learning-based productivity predictions
- Web interface
